Para utilizar la aplicación por primera vez hay que ejecutar el archivo "levantarservicios.sh".
Esto debería levantar todos los contenedores y servicios necesarios para que funcione la aplicación. 
Para el resto de ocasiones en las que se quiera utilizar la aplicación, simplemente bastará con ejecutar "docker-compose up" en el caso de que los contenedores no estén corriendo automáticamente.
